# delayedgram

## Goal

Two folders:

**Config:**
- Default upload delay (1 day)
- Default upload time (6:00am)

**Processed:**

Contains uploaded posts.

**Unprocessed:**

- Multiple folders (smallest number = soonest)
- Each folder contains another folder with images
- Contains yaml file with upload info
  - Title
  - Description
  - ? Location
  - ? Upload time
